# 📋 WEALTHOS 설치 단계별 가이드

이 문서는 기술에 익숙하지 않은 분들도 따라할 수 있도록  
스크린샷 설명과 함께 단계별로 설명합니다.

---

## ⏱️ 총 소요시간: 약 5분

---

## 1단계 — 구글 스프레드시트 생성

1. 브라우저에서 **[sheets.new](https://sheets.new)** 접속
   - 또는 구글 드라이브에서 `+ 새로만들기` → `Google 스프레드시트`
2. 제목을 클릭해 **"WEALTHOS 재산관리"** 로 변경 (선택사항)

---

## 2단계 — Apps Script 편집기 열기

1. 상단 메뉴바에서 **`확장 프로그램`** 클릭
2. 드롭다운에서 **`Apps Script`** 클릭
3. 새 탭에 편집기가 열립니다

> 💡 처음 열면 `function myFunction() {}` 라는 기본 코드가 있습니다

---

## 3단계 — 코드 붙여넣기

1. GitHub의 **`Code.gs`** 파일 열기
2. 파일 내용 **전체 선택** (`Ctrl+A` / `Cmd+A`)
3. **복사** (`Ctrl+C` / `Cmd+C`)
4. Apps Script 편집기로 돌아가기
5. 기존 내용 **전체 선택 후 삭제**
6. 복사한 코드 **붙여넣기** (`Ctrl+V` / `Cmd+V`)
7. **저장** (`Ctrl+S` / `Cmd+S` 또는 💾 아이콘)

---

## 4단계 — 스프레드시트로 돌아가기

1. Apps Script 탭 **닫기** (또는 스프레드시트 탭으로 이동)
2. 스프레드시트 **페이지 새로고침** (`F5` 또는 `Ctrl+R`)
3. 잠시 후 상단 메뉴에 **`💰 WEALTHOS`** 가 나타납니다

> ⚠️ 메뉴가 안 보이면 10~20초 기다린 후 다시 새로고침

---

## 5단계 — 초기화 실행

1. 상단 메뉴 **`💰 WEALTHOS`** 클릭
2. **`🚀 시스템 초기화 (최초 1회)`** 클릭
3. 권한 요청 팝업이 나타납니다:

```
⚠️ 경고: Google에서 이 앱을 확인하지 않았습니다
```

4. **`고급`** 클릭 → **`WEALTHOS(으)로 이동`** 클릭
5. 권한 목록 확인 후 **`허용`** 클릭

> 🔒 **안전합니까?**  
> 네. 이 스크립트는 본인의 스프레드시트만 수정합니다.  
> 외부로 데이터를 전송하지 않으며, 코드가 100% 공개되어 있습니다.

6. 초기화 완료 팝업 확인 → `확인` 클릭
7. 🎉 **완료!**

---

## 6단계 — 나의 데이터 입력

### ⚙️ 설정 시트
- 사용자 이름 변경
- 카테고리별 월 예산 금액 입력

### 🏦 자산 시트
현재 보유 자산을 입력합니다:
- 샘플 데이터를 지우고 본인 자산 입력
- **평가금액** 열만 업데이트하면 수익률 자동 계산

| 자산명 | 유형 | 기관 | 평가금액 |
|---|---|---|---|
| 국민은행 통장 | 예금/현금 | 국민은행 | 8,500,000 |
| 삼성전자 주식 | 주식/ETF | 키움증권 | 5,200,000 |

### 💳 부채 시트
대출 정보 입력:
- 원금, 잔액, 금리, 월 상환액 입력
- 만기일 입력하면 남은 기간 자동 표시

### 📈 투자 시트
주식/ETF 입력:
- 종목명, 수량, 평균단가, **현재가** 입력
- 현재가만 주기적으로 업데이트하면 수익률 자동 계산

### 🎯 목표 시트
재무 목표 설정:
- 목표명, 목표금액, 현재금액, 목표일 입력
- 월 필요 저축액, 달성률, 예상 달성일 자동 계산

---

## 7단계 — 가계부 사용

### 매일 거래 기록
`💰 WEALTHOS` → `➕ 거래 빠른 입력` 클릭

또는 `📒 가계부` 시트에서 직접 입력:
1. 날짜 입력
2. 유형 선택 (드롭다운: 수입/지출)
3. 카테고리 선택 (드롭다운)
4. 내용/가맹점 입력
5. 금액 입력

### 자동 분류 사용
1. 내용을 입력한 후
2. `💰 WEALTHOS` → `🔄 카테고리 자동 분류`
3. 가맹점명 키워드 자동 인식

---

## 8단계 — 대시보드 확인

`💰 WEALTHOS` → `📊 대시보드 새로고침`

다음 정보를 한눈에 확인:
- 순자산 (총 자산 - 총 부채)
- 이번 달 수입/지출/저축률
- 카테고리별 지출 TOP 5 + 예산 대비 현황
- 목표 달성 진행률

---

## 🔄 다른 사람과 공유하기

### 방법 1: 링크 공유 (보기 전용)
1. 오른쪽 상단 `공유` 버튼 클릭
2. `링크가 있는 모든 사용자` 선택
3. 권한: `뷰어` 선택
4. 링크 복사 후 공유

### 방법 2: GitHub으로 코드 공유 (다른 사람이 설치)
1. GitHub에서 이 저장소를 Fork
2. 본인 GitHub 링크 공유
3. 상대방이 위의 1~5단계를 따라 설치

### 방법 3: 스프레드시트 복사본 공유
1. `파일` → `사본 만들기`
2. 사본을 `편집자` 권한으로 공유
3. 상대방이 받아서 본인 데이터 입력

> ⚠️ 재무 데이터는 민감합니다. **개인용으로 각자 복사해서 사용하는 것을 권장**합니다.

## ❗ 문제 해결

| 문제 | 해결 방법 |
|---|---|
| `💰 WEALTHOS` 메뉴가 안 보임 | 페이지 새로고침 후 20초 대기 |
| 권한 오류 발생 | 스크립트 권한을 다시 허용 (3단계 반복) |
| 대시보드가 비어있음 | 자산/가계부 데이터 입력 후 새로고침 |
| 자동 분류가 안 됨 | 카테고리 시트 C열에 키워드 추가 |
| 수식이 깨짐 | 시트 삭제 후 시스템 초기화 재실행 |
